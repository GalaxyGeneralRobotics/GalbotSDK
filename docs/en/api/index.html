    ControlStatus

    机器人执行状态枚举，用于表示动作或操作的执行结果。

    枚举值:
    • SUCCESS : 执行成功
    • TIMEOUT : 执行超时
    • FAULT : 发生故障无法继续执行
    • INVALID_INPUT : 输入参数不符合要求
    • INIT_FAILED : 内部通讯组件创建失败
    • IN_PROGRESS : 正在运动中但未到位
    • STOPPED_UNREACHED : 已停止但未到达目标
    • DATA_FETCH_FAILED : 数据获取失败
    • PUBLISH_FAIL : 数据发送失败
    • COMM_DISCONNECTED : 连接失败
